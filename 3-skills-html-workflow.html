01

taste-skill

github.com/Leonxlnx/taste-skill
64K+ ★
Generate · 反平庸审美框架

号称 "The Anti-Slop Frontend Framework for AI Agents"。它不是模板库，而是给 AI 注入审美判断力——编码前先推断行业、情绪与版式，再避开 Hero 三段式、居中卡片堆叠、emoji 图标等一眼假的"AI Slop"。

  • Brief Inference 推理：先想清楚"给谁看、什么调性"再动手
  • 反模式清单：主动规避千篇一律的渐变 Hero 与默认主题直出
  • 一次性输出完整可运行代码，覆盖移动端到桌面端响应式
02

html-edit

github.com/acelit/html-edit
Edit · 可视化改稿

给任意现成的 HTML 页面注入一个悬浮的编辑按钮：平时透明淡出、悬停才浮现，全屏时自动隐藏。点开后整页文字、图片、表格都变成富文本编辑器，改完保存即可导出新的 HTML。

  • 富文本操作：加粗 / 标题 / 列表 / 颜色 / 对齐 / 插图 / 链接
  • 表格可增删行列，所见即所得，不用回头改代码
  • 导出的 HTML 仍带编辑按钮，可反复二次编辑
03

html-open

github.com/acelit/html-open
Deploy · 一键发布

把离线 HTML 文件一键发布为 GitHub Pages 公开链接。给出一个本地 .html，脚本自动推送到托管仓库，返回形如 https://…github.io/…/文件名.html 的公网访问地址。

  • 本地文件直接变公开链接，方便分享与协作评审
  • 基于 GitHub Pages 托管，稳定且免费
  • 支持自定义文件名 / 唯一哈希 / 覆盖更新
